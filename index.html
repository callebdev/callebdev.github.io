Here are some of my personal projects

AlugAqui

AlugAqui is an application that aims to facilitate the dissemination, promotion and research of properties to rent in Mozambique. It offers diversified options such as searching for properties on the map, filters and a chat system that facilitates the properties' owners to answer some users' doubts regarding their properties. The application is currently only available for Android OS. It can be found on Google Play, it is completely FREE and simple to use.

Watch-Next

Watch Next is an android application built using the MVVM pattern, RXJava, Databinding and Room. Watch-Next basically displays a list of TV shows, allows you to fuzzy search for a TV show and create a watch list. It makes requests to EPISODATE API and parses the JSON response to display the TV Shows.
